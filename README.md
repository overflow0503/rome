#### 项目结构：
* admin-ui : 预留的可能存在的控制中台
* cache: 缓存相关
* common：通用类
* Config：配置
* generator：代码生成
* manage：核心业务类
  * dao | searvice | webapp
* parent：pom管控
* scheduling：预留的可能定时中心
* security：安全
